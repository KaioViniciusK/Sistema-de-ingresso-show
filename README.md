Sistema de Ingressos para Shows – Projeto em C

Este projeto consiste no desenvolvimento de um sistema em linguagem C para gerenciamento de ingressos de shows. 
O programa permite que usuários visualizem eventos disponíveis, realizem compras de ingressos e pesquisem shows por nome ou categoria. 
Além disso, há uma área exclusiva para administradores, com funções de gerenciamento completo dos shows cadastrados.
